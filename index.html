Форматы консультаций

Без обещаний конкретного исхода. В фокусе: анализ запроса, возможные сценарии и рекомендации по вашим действиям.

Личные отношения

Разбор сложной коммуникации, эмоционального состояния и возможных шагов, которые вы можете предпринять без давления на других людей.

Сравнение нескольких вариантов решения, фиксация плюсов и рисков, подготовка понятного плана действий на ближайший период.

Эмоциональный фокус

Помощь в наведении внутренней ясности, приоритетов и границ, чтобы принимать решения в более спокойном состоянии.

1. Какие данные собираются

Контактные данные, которые пользователь передаёт добровольно (имя, телефон, адрес электронной почты), а также технические данные браузера (cookie, IP, параметры устройства).

2. Для чего используются данные

Для обратной связи, записи на консультацию, сопровождения клиента и улучшения качества работы сайта.

3. Срок и защита хранения

Данные хранятся только необходимый срок и обрабатываются с ограничением доступа. По запросу пользователя данные могут быть уточнены или удалены при отсутствии законных ограничений.

4. Передача третьим лицам

Передача возможна только в случаях, предусмотренных законом, либо сервисным подрядчикам в объёме, необходимом для технического функционирования.
